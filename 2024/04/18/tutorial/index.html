本教程手把手介绍了如何使用Hexo+GitHub 搭建个人博客。

软件下载

Hexo 基于 Node.js，搭建过程中还需要使用 npm 和 git，因此先搭建本地操作环境，安装 Node.js 和 Git。

  • Node.js：https://nodejs.org/zh-cn
  • Git：https://git-scm.com/downloads

下载 Node.js 和 Git 程序并安装

连接Github

通过Git Bash Here，设置用户名和邮箱：

创建 SSH 密匙

添加密匙
进入 [C:\Users\用户名.ssh] 目录（要勾选显示“隐藏的项目”），用记事本打开公钥 id_rsa.pub 文件并复制里面的内容。

登陆 GitHub ，进入 Settings 页面， SSH and GPG keys中新建 SSH key 并保存。

接着创建一个Github Pages 仓库

本地安装 Hexo 博客程序
